# LINK 컨설팅 진단

보험 컨설턴트를 위한 **LINK 프로세스 자가진단 웹앱**입니다.
5개 질문을 통해 L·I·N·K 4단계 중 가장 보완이 필요한 단계를 진단하고, 교육 전 집중 포인트를 안내합니다.

## LINK 4단계

| 단계 | 이름 | 설명 |
|------|------|------|
| **L** | 후킹 (Hooking) | 고객과의 첫 만남에서 보험 대화를 자연스럽게 여는 단계 |
| **I** | 진단 (Investigation) | 고객 상황을 파악하고 개인정보를 확보하는 단계 |
| **N** | 설계 (Navigation) | 담보 우선순위를 정하고 제안서를 구성하는 단계 |
| **K** | 클로징 (Closing) | 망설이는 고객의 결정을 이끌어내는 단계 |

## 화면 구성

### 1. 메인 화면
- LINK 브랜드 타이틀과 진단 시작 버튼
- L → I → N → K 스텝 인디케이터 표시

### 2. 정보 입력
- **지점명**: 텍스트 입력
- **사번**: 숫자 7자리 유효성 검증

### 3. 진단 질문 (Q1~Q4)
각 질문은 실제 영업 현장의 상황을 제시하고, 해당 상황에서의 자신의 수준을 3단계로 응답합니다.

- **Q1 (L단계)**: 첫 만남에서 보험 대화를 여는 능력
- **Q2 (I단계)**: 니즈 환기 후 개인정보 확보 능력
- **Q3 (N단계)**: 담보 설계 시 우선순위 판단 능력
- **Q4 (K단계)**: 클로징 단계에서 결정을 이끄는 능력

각 선택지의 점수:
- `1` = 흐름이 잡혀있다 (양호)
- `2` = 가끔 고민된다 (보통)
- `3` = 늘 어렵다 (약점)

### 4. 교육 목표 선택 (Q5)
- 복수 선택 가능
- 오늘 교육에서 가장 배우고 싶은 영역을 선택

### 5. 결과 화면
- 진단 결과와 교육 목표를 종합하여 **오늘 집중할 단계** 1개를 안내
- L·I·N·K 4단계 시각 요약 제공

### 6. 관리자 대시보드
- URL에 `?admin` 쿼리 파라미터 추가 시 진입
- 총 응답 수, 단계별 병목 분포 차트
- CSV 다운로드 기능

## 결과 판정 로직

1. Q1~Q4에서 점수가 2 또는 3인 단계를 **약점(weak)**으로 분류
2. Q5(교육 목표)에서 선택한 단계 중 약점과 겹치는 것이 있으면 해당 단계를 결과로 반환 (본인 인지 일치)
3. 겹치는 것이 없으면:
   - Q5 선택이 있는 경우 → Q5 첫 번째 선택 반환 (양호하지만 더 배우고 싶은 영역)
   - Q5 선택이 없는 경우 → L → I → N → K 순으로 첫 약점 반환
4. 약점이 전혀 없으면 → L단계를 기본값으로 반환 (전 단계 양호)

## 기술 스택

- **순수 HTML/CSS/JavaScript** — 프레임워크 없는 싱글 파일 구성
- 모바일 최적화 (max-width 480px)
- 슬라이드 전환 애니메이션 (CSS keyframes)
- **localStorage**를 사용한 응답 데이터 저장
- CSV 내보내기 (Blob + BOM 처리로 한글 호환)

## 실행 방법

`index.html` 파일을 브라우저에서 직접 열면 됩니다. 별도의 빌드나 서버가 필요 없습니다.

```
# 관리자 페이지 접근
index.html?admin
```

## 데이터 저장

응답 데이터는 브라우저의 `localStorage`에 `link_resp_{timestamp}` 키로 저장됩니다.

저장되는 항목:
- `branch` — 지점명
- `empId` — 사번
- `answers` — Q1~Q4 응답 배열 `[1~3, 1~3, 1~3, 1~3]`
- `q5` — Q5 선택 배열 `["L", "I", ...]`
- `result` — 최종 결과 단계 `"L" | "I" | "N" | "K"`
- `timestamp` — ISO 형식 타임스탬프
