# LINK 컨설팅 진단 × 실적 연계 분석 보고서

> 218명 매칭 (8개 지점, 2-3월 평균 실적 기준)

## 핵심 발견 요약

### 발견 1: 실적을 가장 크게 가르는 단일 문항은 Q7 "생각해볼게요" 대응
- Q7(결정 유도)의 실적 상관 **r = -0.308** (경력 통제 후) → 8개 문항 중 압도적 1위
- 경력이 같아도 이 문항을 어렵게 느끼는 사람과 쉽게 느끼는 사람의 실적 격차가 가장 큼
- Q7 + Q5(담보 우선순위)를 **둘 다** 어렵게 느끼면 실적 타격 최대 (r = -0.334)

### 발견 2: 경력 단계마다 "실적을 가르는 병목"이 다르다
| 경력 | 실적 상·하위 최대 격차 단계 | 최대 격차 문항 | 해석 |
|------|---------------------------|--------------|------|
| 6개월 이내 | **N(설계)** +0.56 | Q6 보험료 부담시 플랜 제시 | 설계력이 관문 |
| 7~12개월 | **I(진단)** +0.61 | Q5 담보 우선순위 | L·I·N 골고루, K는 차이 없음(=다 어려움) |
| 1~2년 | **I(진단)** +0.55 | Q3 문제 인식시키기 | 진단 스킬이 갈림돌 |
| 2~10년 | **L(연결)** +0.88 ★ | Q1 첫만남 화법 (차이 **1.19**) | 대면 기회 창출이 핵심 |
| 10년+ | **K(해결)** +0.55 | Q4 인증번호 확보 (차이 0.68) | 베테랑도 K + 디지털 분석 |

→ **신인은 N·I를 못 해서, 중견은 L을 못 해서, 베테랑은 K를 못 해서 실적이 갈린다.**
→ 전 경력대에 같은 교육을 하면 안 됨. 경력별 맞춤 교육이 필요.

### 발견 3: "눈먼 자신감" 그룹 — 자기인식 4사분면
| 유형 | 인원 | 실적 | 난이도 | 특징 |
|------|------|------|--------|------|
| [A] 실력+자신감 | 64명(29%) | 135만 | 2.13 | 멘토 후보. 강진·은계·고양에 집중 |
| [B] 실력O 자신감X | 45명(21%) | 104만 | 3.42 | 실력 있는데 불안. 자신감 코칭 필요 |
| [C] **자신감O 실력X** ⚠️ | 40명(18%) | 50만 | 2.20 | **가장 위험**. 용인11명·성남7명 집중 |
| [D] 둘다 부족 | 69명(32%) | 50만 | 3.54 | 교육 효과 기대 가능 (본인이 알고 있음) |

- [C]그룹은 실적이 [D]와 같은 수준인데 **본인은 괜찮다고 생각함**
- [C]그룹 남성 비율 28%로 전체(23%)보다 높음 → 남성에서 과잉 자신감 경향
- **용인SFP지점에 [C]가 11명(39%)** — 지점 전체에 자기진단 직면 프로그램 필요
- **성남지점도 [C]가 7명(39%)** — 동일 이슈

### 발견 4: K(해결)은 "보편적 어려움"이지만, K 안에서도 갈린다
- K 난이도 1~2점: 실적 119만 / K 난이도 4~5점: 실적 68만 → 75% 차이
- **K를 동일하게 통제해도** N(설계)이 실적을 추가 설명 (r = -0.207)
- → K가 어려운 건 당연하지만, **같은 K 난이도 내에서 N(설계)이 갈림**
- Q7(결정유도) + Q5(담보우선순위) 조합이 실적 예측력 최고 (r = -0.334)

### 발견 5: 균형 잡힌 사람이 실적이 높다 — 그리고 난이도 낮을수록 격차 확대
| 난이도 수준 | 균형잡힌 | 편향된 | 차이 |
|------------|---------|--------|------|
| 높음 (≥3.0) | 75만 | 67만 | +9만 |
| 보통 (2.0~3.0) | 96만 | 89만 | +7만 |
| **낮음 (<2.0)** | **127만** | **91만** | **+36만** |

- 난이도가 낮은(=실력 좋은) 그룹에서 균형/편향 격차가 폭발
- → 실력이 쌓일수록 **올라운드 밸런스**가 실적의 결정적 차이를 만듦
- 최고 실적 조합: "난이도 낮음 + 균형잡힌" = 평균 **116만**
- 최저 실적 조합: "난이도 높음 + 중간 편차" = 평균 **62만**

### 발견 6: Q4(인증번호 확보)의 숨겨진 의미 — 10년+ 베테랑의 디지털 격차
- 전체 평균 2.49로 가장 쉬운 문항 → 별로 안 중요해 보임
- 그런데 **10년+ 베테랑에서 Q4 4~5점이 31%** (6개월 이내 24%보다 높음)
- 10년+에서 Q4 실적 상관 **-0.245** → 베테랑 중 디지털 도구(간편분석) 적응 못 하는 사람이 실적에서 갈림
- 1~2년차에서는 Q4가 이미 쉬워짐(2.05) → 디지털 네이티브 세대는 문제 없음

### 발견 7: 같은 경력에서 FP가 실적 높지만, 10년+ 구간에서 역전
| 경력 | FP 실적 | SFP 실적 | 차이 | FP 특이점 |
|------|---------|----------|------|----------|
| ≤6개월 | 83만 | 67만 | FP +16만 | L·I를 SFP보다 더 어려워함 |
| 2~10년 | 93만 | 86만 | FP +7만 | L·I·N 모두 더 어려워함 |
| **10년+** | **103만** | **146만** | **SFP +43만** | SFP가 I(진단)만 더 어려움 |

- 10년+ 베테랑 SFP가 FP보다 실적 43만 높음
- SFP 베테랑은 진단(I)만 보강하면 되는 상태, FP 베테랑은 전반적으로 정체

### 발견 8: 신인 6개월 이내 — 성별 격차가 가장 큰 구간
| | 남성(21명) | 여성(34명) |
|---|-----------|-----------|
| 실적 | **83만** | **63만** |
| 난이도 | 3.07 | 3.47 |
| N(설계) | 3.07 | **3.63** |
| K(해결) | 3.40 | **3.85** |

- 6개월 이내 여성 신인이 N(설계)·K(해결)을 극도로 어려워함
- 실적도 남성보다 20만 낮음
- **하지만 7~12개월이 되면 여성이 역전** (남 63만 < 여 81만)
- → 여성 신인은 초기 6개월 집중 지원 시 효과 극대화 가능

## 교육 기획 시사점

### 1. 전 경력대 공통 — Q7(결정유도)이 실적 예측 1위
- "생각해볼게요" 대응 화법을 모든 교육 과정의 핵심으로

### 2. 경력별 맞춤 교육
- **6개월 이내**: N(설계) + K(해결) → "플랜 제시 + 클로징" 패키지
- **7~12개월**: I(진단) + N(설계) → "분석지 활용 + 담보 우선순위"
- **1~2년**: I(진단) → "고객 문제 인식 화법"
- **2~10년**: L(연결) → "첫 만남 후킹 화법" (이 구간에서 Q1 격차 1.19로 최대)
- **10년+**: K(해결) + Q4(디지털 분석 도구) → "클로징 + 간편분석지"

### 3. "눈먼 자신감" 지점에 자기직면 프로그램
- 용인SFP·성남: 설문 결과와 실적을 대면시키는 워크숍
- 단순 교육보다 인식 전환이 먼저

### 4. N+K 복합 약점이 실적 타격 최대
- Q5 + Q7 조합이 r = -0.334 (가장 강한 부정적 상관)
- → 설계와 클로징을 연결하는 통합 교육 설계
